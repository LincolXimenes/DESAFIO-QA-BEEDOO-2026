# Vídeos - Evidências dos Testes

Esta pasta armazena todas as gravações de tela coletadas durante a execução dos testes da aplicação de cadastro de cursos.

## 📁 Organização dos Arquivos

### Nomenclatura Padrão
Os arquivos seguem o padrão: `[TIPO]_[DESCRIÇÃO]_[DATA].[EXT]`

**Exemplos:**
- `BUG001_lista_vazia_demo_20260309.mp4` - Demonstração do bug da lista vazia
- `FLOW_cadastro_completo_20260309.mp4` - Fluxo completo de cadastro
- `TEST_execucao_casos_principais_20260309.mp4` - Execução dos principais casos

### Tipos de Gravação

#### 🐛 Demonstração de Bugs (BUG)
- Gravações mostrando reprodução de bugs
- Passos completos para reproduzir o problema
- Formato: `BUG[ID]_[descrição]_[data].mp4`

#### 🔄 Fluxos Completos (FLOW)
- Gravações de fluxos end-to-end
- Navegação completa pela aplicação
- Formato: `FLOW_[descrição]_[data].mp4`

#### 🧪 Execução de Testes (TEST)
- Execução sistemática de casos de teste
- Validação de resultados esperados
- Formato: `TEST_[descrição]_[data].mp4`

#### ⚠️ Problemas Identificados (ISSUE)
- Gravações de comportamentos inesperados
- Inconsistências na interface
- Formato: `ISSUE_[descrição]_[data].mp4`

#### 📱 Responsividade (RESP)
- Testes em diferentes tamanhos de tela
- Comportamento mobile/tablet
- Formato: `RESP_[dispositivo]_[data].mp4`

## 🎬 Diretrizes para Gravação

### Configurações Técnicas
- **Resolução:** 1280x720 (HD) ou superior
- **Frame rate:** 30 FPS mínimo
- **Formato:** MP4 (melhor compatibilidade)
- **Duração:** Máximo 5 minutos por vídeo
- **Audio:** Incluir narração quando necessário

### Conteúdo
- **Início:** Mostrar URL e data/hora
- **Contexto:** Explicar o que será testado
- **Execução:** Demonstrar passos de forma clara
- **Resultado:** Mostrar o resultado obtido
- **Conclusão:** Resumir o que foi validado

### Qualidade
- Movimento de mouse suave e deliberado
- Pausa entre ações para clareza
- Destacar elementos importantes
- Evitar distrações na tela

## 📝 Ferramentas Recomendadas

### Windows
- **OBS Studio** (gratuito, profissional)
- **ScreenRec** (gratuito, simples)
- **Camtasia** (pago, recursos avançados)
- **Gravador de Passos** (nativo Windows)

### Chrome/Edge
- **Extensões de gravação** de tela
- **DevTools** para gravação de performance

## 🎯 Casos de Uso para Vídeos

### 1. Demonstração de Bugs
- Reproduzir bugs de forma clara
- Mostrar comportamento atual vs esperado
- Facilitar entendimento pelos desenvolvedores

### 2. Validação de Casos de Teste
- Documentar execução de cenários complexos
- Registrar evidências de comportamento correto
- Facilitar revisões de teste

### 3. Análise de Usabilidade
- Identificar problemas de experiência
- Demonstrar fluxos confusos ou ineficientes
- Sugerir melhorias baseadas no uso real

### 4. Documentação de Processos
- Criar guias visuais para testes
- Treinar outros testadores
- Manter histórico de validações
